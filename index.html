What is Coding?

Coding is the process of writing instructions in a programming language so that a computer can execute specific tasks or solve problems. These instructions, called programs, are translated into machine-readable binary (0s and 1s) that hardware can understand. It is the foundation of software development, powering everything from mobile apps to artificial intelligence systems.

Types of coding languages

1. HTML

HTML stands for Hypertext Markup Language. It is the standard markup language used for creating and designing web pages and web applications, HTML defines the structure of web pages by using a variety of elements and tags that tell the browser how to display the content.

Audio

The audio tag is used to embed sound content in a webpage, such as music or sound effects. It supports multiple file formats and can be controlled using attributes such as controls, autoplay, muted, and loop.

        The following code used is:

        <audio controls autoplay loop>
            <source src="music/Golden - Vibe Tracks copy.mp3">
        </audio>

In the event that your first audio file fails to load, you can provide a backup file by adding a second source tag with a different file format, such as WAV. The browser will automatically resort to the next available source.

        <audio controls autoplay muted loop>
            <source src="music/Golden - Vibe Tracks copy.mp3" type="audio/mpeg">
            <source src="music/Golden - Vibe Tracks copy.wav" type="audio/wav">
        </audio>

Favicon

A favicon is a small icon that represents your webpage, typically displayed on the browser tab. It is defined in the head section of your code.

To use a favicon, you need an image that is at least 96 by 96 pixels. You can resize your image using any image editor such as Paint.

A favicon is created using the <link> tag with the rel, type, and href attributes as follows.

        link rel="icon" type="image/jpg" href="image/favicon.jpg">

<span> and <div> tags

The <span> and <div> tags are mainly used for styling with CSS.

The <div> tag is a block container that takes up the entire width of the page, like a shelf.

The <span> tag is an inline container that is only as big as the text inside it, like a sticky note. Both tags can be applied to individual words, phrases, or entire sections. Below are some examples.
